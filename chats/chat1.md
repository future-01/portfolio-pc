## User

あなたは、家族向けの安心感と、作品展示としての洗練さを両立することが得意な、トップレベルのWebデザイナー兼UI/UXデザイナーです。

旅行先や家族で一緒に撮った写真をウェブ上で見られる、限定公開の写真ポートフォリオサイトをデザインしてください。
主な閲覧環境はパソコンではなくスマホです。必ずスマホファーストで設計してください。

## サイトの前提条件

- 公開範囲: 家族・親しい人だけの限定公開
- 主な閲覧者: 家族
- サイトの目的: 半分は家族向けの思い出共有、半分は作品展示
- 掲載する写真:
  - 旅行の風景写真
  - 記念撮影などの人物写真
- 初期掲載枚数: 少なくとも100枚
- 今後の増加想定: 年十数枚ずつ増える
- 分類方法:
  - 第一優先: 旅行先別
  - 補助的に: 年別
- 写真ごとに表示したい情報: 日付
- 必須でほしい要素: アルバム一覧、写真の拡大表示
- 任意要素:
  - ダウンロード(できればフルサイズ版)
  - 動画掲載
  ただし、サイトが重くなったり、UIが複雑になったり、限定公開・顔写真ありという条件に対して相性が悪い場合は、省略または代替案を提案してよい
- 顔写真あり
- UI文言まで作成したい
- いったん、ある程度目に見える形まで具体化したい

## デザインの方向性

- かっこよさ 3 : 温かみ 7
- ベージュ系
- カフェ風
- モダン
- 高級感のある控えめな演出のみ
- 操作が簡単で、写真が大きく見えることを重視
- 余白が少なすぎるデザインは避ける

## 実現したい印象

- スタイリッシュでかっこいい
- ただし冷たすぎない
- ファミリー層が見ても安心感がある
- 万人受けする温かみがある
- 写真が主役として映える
- 親しみやすいが、野暮ったくない
- 作品展示としても成立する上品さがある

## 避けたい方向性

- 余白不足で窮屈な画面
- 無機質すぎるミニマルデザイン
- 暗すぎる・重すぎる高級路線
- 子ども向けすぎるポップなデザイン
- ボタンや文字が小さく、スマホで押しづらいUI
- 情報量が多すぎて写真を邪魔する構成
- ベージュ系なのにコントラスト不足で見づらい設計

## 進め方

最初に追加質問を返すのではなく、まずこの条件で完成度の高い第1案を提示してください。
不足情報があっても制作を止めず、合理的な仮定を置き、その仮定は明示してください。

## 提案時のルール

- スマホファーストを最優先にすること
- 375〜430px程度のスマホ幅を主基準として考えること
- 写真が大きく、美しく見えることを優先すること
- 片手操作でも迷いにくい導線にすること
- 写真を邪魔しないUIにすること
- 家族向けの安心感を大事にしながら、作品展示としても見栄えするデザインにすること
- 温かみは出してよいが、生活感が出すぎて野暮ったくしないこと
- モダンさは出してよいが、冷たくしすぎないこと
- 抽象論ではなく、実際の画面に落とし込める具体性で示すこと
- 必要であれば、複数案を比較しつつ、最終的に最も適した1案を推奨すること
- UI文言は日本語で、やさしく自然で、家族でも使いやすい表現にすること

# Memoria — 家族のための写真ポートフォリオ サイト 第1案

「かっこよさ3 : 温かみ7」を、ベージュ × 焙煎ブラウン × 明朝体で構成した、限定公開の家族写真集です。デザインキャンバスに **設計指示書 6点 + 実機モック 11画面** を並べました。

## 仮定したこと（明示）
- 招待者: 4名（管理者1 + 家族3）／ 利用者は「あやか」さん
- 初期掲載 110枚を 7アルバムに分散（旅行先別が一次、年別が補助）
- フォントは Google Fonts の Shippori Mincho / Noto Sans JP / Cormorant Garamond
- 写真は SVG プレースホルダー（実写差し替え前提）

## 推奨を1案に絞ったもの
| 論点 | 推奨 | 理由 |
|---|---|---|
| 配色 | **A. Sand** | 家族の肌色を邪魔しない / 温かみと作品感の中間 |
| アルバム一覧 | **2列グリッド + 最新ヒーロー** | 写真が主役、片手操作で迷わない |
| ダウンロード | 採用 / 家族のみ・長押し格納 | 顔写真流出リスクを最小化 |
| 動画 | 採用 / 30秒上限・自動再生なし | サイトを軽く・誤再生防止 |
| 第一分類 | 旅行先別アルバム | 年に十数枚増えても破綻しない |

## 主な画面
1. **ロック** — 合言葉、ぼかし背景、温度のある暗色  
2. **ホーム** — 最新ヒーロー + 2列アルバム + 横スクロール年別  
3. **アルバム一覧（リスト案）** — 比較用  
4. **アルバム詳細** — 日付グループ + 1枚目だけフルワイド  
5. **ライトボックス** — 黒背景 + グラスモーフィズムのキャプション  
6. **年別 / さがす / 設定** — 補助動線  
7. **配色 3案比較** — 同じホームで Sand/Latte/Linen 並列

## 次にいただきたいご判断
- 配色は **Sand** で進めて良いか（Latte/Linen のどちらかに寄せる場合は教えてください）
- 動画は採用するか省くか
- 招待者の「家族／親しい人」の権限差をもう少し詰めたいか
